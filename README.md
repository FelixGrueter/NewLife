# new Life

Website fuer das new Life Hausgemeinde-Netzwerk — ein Werk von MANTD.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- GSAP (Animationen via CDN)
- Google Fonts: Cormorant Garamond, DM Sans, Dancing Script
- REST APIs: restcountries.com, zippopotam.us (Kontaktformular)
- Bilder via Unsplash (extern)

## Projektstruktur

```
index.html              Hauptseite (alles inline)
```

## Lokal starten

```bash
npx serve .
```
